Adding an Account

# passwd joeracker
Changing password for user joeracker.
New password:
BAD PASSWORD: it is based on a dictionary word

Getting Account Information

# id joeracker
uid=501(joeracker) gid=502(joeracker) groups=502(joeracker)

# passwd -S joeracker
joeracker PS 2016-06-08 0 99999 7 -1 (Password set, SHA512 crypt.)

# chage -l joeracker
Last password change				: Jun 09, 2016
Password expires					: never
Password inactive					: never
Account expires						: never
Minimum number of days between password change		: 0
Maximum number of days between password change		: 99999
Number of days of warning before password expires	: 7

Files in Play

  • /etc/passwd
  • /etc/shadow
  • /var/log/secure

/etc/passwd

  • Account username
  • password - not really
  • User ID
  • Group ID
  • Comment field
  • Home directory
  • User shell

/etc/shadow

  • Username
  • Password
  • Last password change
  • Minimum
  • Maximum
  • Warn
  • Inactive
  • Expire

/var/log/secure

Contains some of the following
  • User attempts to connect
  • A user gaining sudo access
  • User disconnects
